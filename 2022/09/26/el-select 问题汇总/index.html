一、el-select 选择器图片回显问题

1.问题描述
使用 select 选择器时，下拉显示图片，选择之后并回显图片

2.问题探索
首先使用了 element-ui 的 select 选择器，使用过程中，只能是下拉框有图片，但是不能回显，之后尝试了通过 $ref 来获取实例，进行操作，出现的问题为：select 选择器图片能正常显示，但多了一个输入框，通过打印检查了一下，他是在 div 里面的，删除 input 之后图片也会消失，然后只能回显一个，再次点击此输入框就会显示 base64 和图片地址，图片就没了。以上尝试失败。

3.解决方法
使用 vue-multiselect

4.代码演示

  • npm install vue-multiselect@2.1.0
  • 全局引入，千万不要忘记引入 css 样式！！！

5.效果

    vue-multiselect 官网

二、原生 select 中 option 长度

2.解决方法
刚开始是从option的样式区修改 但怎么都修改不掉 查之后说不能修改建议改成 div ul li 的样式 但由于不好改想出来另外一个方法 就是通过 动态渲染写三元表达式
