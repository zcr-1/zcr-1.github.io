一、父子传值数组默认值的问题

1.问题描述
父向子以数组的形式进行传值，默认值写成的null，根据数据是否存在显示或者隐藏，后台报错，没有属于这个长度

2.控制台
Error in render: “TypeError: Cannot read properties of null (reading ‘length’)”

3.解决办法
由原来的 default: null 改成 default: () => [] 工厂形式
